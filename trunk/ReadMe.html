SVN 2012 FIRST Robotics Game Documents

This is an unofficial collection of the documents for the 2012 FIRST Robotics competition. The collection is maintained by Lynbrook High School Robotics FIRST Team 846.

The SVN repository provides a quick way to obtain an off-line copy of the rules. The documents are automatically collected several times daily from the official FIRST website and committed to this repository.

SVN Repository Location:
http://frc2012-gamedocs.googlecode.com/svn/trunk/

SVN checkout command:
svn checkout http://frc2012-gamedocs.googlecode.com/svn/trunk/ frc2012-gamedocs-read-only

The file fileList.txt in each directory contains the list of files embedded in each html web page. Some of these files are not visible on the official FIRST manual page, and their contents may not be current.

By checking the SVN log, you may review the revision history of every document.

Remember, this is an unofficial non-sanctioned collection of the game rules. The official rules are located at www.usfirst.org.

Please direct comments or questions about this repository to FRC12_rules@lynbrookrobotics.com.
